# Imarisha Jamii Centre | Adult Education School

A premium, high-conversion website for Imarisha Jamii Centre, an adult education institution based in Umoja, Nairobi. Built with a focus on accessibility, professional branding, and an optimized admissions funnel.

## Project Overview

Imarisha Jamii Centre provides a second chance for adult learners through flexible KCSE completion programs, bridging courses, and professional skills training. This platform serves as the primary digital home for prospective students to explore programs and apply online.

## Core Features

- **Admissions Funnel** - Complete application flow with multi-step forms and validation.
- **Program Catalog** - Detailed breakdown of academic and professional courses.
- **Responsive Design** - Optimized for mobile viewing as many adult learners access via smartphones.
- **SEO Ready** - Semantic HTML and meta configurations tailored for Nairobi-based educational searches.
- **Admin Dashboard** - CMS-ready structure for managing student applications (connected to Sanity).

## Tech Stack

- **React 19** + **TypeScript**
- **Vite** - High-performance build tool
- **Tailwind CSS 3** - Modern styling system
- **GSAP** + **ScrollTrigger** - Cinematic scroll-driven interactions
- **Lenis** - Smooth scroll experience
- **Radix UI** - Accessible UI primitives
- **Lucide React** - Icon system

## Development

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Start Dev Server**
   ```bash
   npm run dev
   ```

3. **Build for Production**
   ```bash
   npm run build
   ```

## Configuration

Site content, contact details, and program information are centrally managed in `src/config.ts`. Key configuration areas include:

- `siteConfig` - Global metadata and brand naming.
- `contactConfig` - Verified phone, email, and location data.
- `programsConfig` - Course descriptions, schedules, and requirements.
- `mainNavLinks` - Global navigation structure.

## Deployment

Optimized for deployment on Vercel with integrated SEO and edge performance.
